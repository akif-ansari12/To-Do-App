# 📝 Advanced To-Do App (Python)

## 📌 Description
A feature-rich command-line To-Do application built using Python.  
It allows users to manage tasks efficiently with priority, due date, and search functionality.

---

## 🚀 Features
- Add new tasks
- Delete tasks
- Mark tasks as complete
- Set priority (Low, Medium, High)
- Add due date
- Search tasks
- Persistent storage using JSON

---

## 🛠️ Technologies Used
- Python
- Object-Oriented Programming (OOP)
- File Handling (JSON)

---

## ▶️ How to Run
1. Clone the repository:https://github.com/akif-ansari12/To-Do-App.git

2. Navigate to folder: cd todo-app
